# Task 1 — JSON Data Formatting

Reads an array of article objects from a `data` variable and prints each one into `#results` in this format:
Article "André Baniwa" (Page ID 6682420) was created at September 13, 2021.

## Usage

Open `index.html` in a browser. No dependencies or build steps.

## Notes

- Dates are formatted using `Date.toLocaleDateString("en-US")` — no external libraries.
- The page also displays its own source code in the `#code` block via `document.getElementById("task_id").text`.
